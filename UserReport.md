# 📋 UserReport - Taima Al-Wani Cyber Academy

## 🏗️ نظرة عامة
تطبيق "أكاديمية تيماء العلواني للأمن السيبراني" هو منصة تعليمية تفاعلية تحاكي الهجمات السيبرانية وتقدم ألعاباً تعليمية (مثل محاكاة المدينة الآمنة Secure City، ولعبة من سيربح المليونير السيبرانية، ومسابقات التشفير، واختبار تحديد المستوى التفاعلي).
يعتمد التطبيق على واجهة أمامية مبنية بـ Vite + React وتنسيقات TailwindCSS v4، وخادم خلفي بدون خادم (Serverless) يعمل على Cloudflare Pages Functions وقاعدة بيانات Cloudflare D1 (SQLite).
