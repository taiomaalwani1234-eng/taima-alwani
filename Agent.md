# 🧠 Agent.md - Taima Al-Wani

## 📊 نظرة عامة
- **نوع المشروع:** تطبيق ويب كامل (Full-stack Web App) مع واجهة خادم خلفي على Cloudflare Pages Functions.
- **اللغة:** TypeScript / TSX
- **الإطار:** React + Vite + TailwindCSS v4 + Cloudflare Pages Functions (D1 Database)
- **نقطة الدخول:** `src/main.tsx` للواجهة الأمامية و `functions/[[route]].ts` للخادم الخلفي.

## ✅ أفضل الممارسات المكتشفة
- **الهجرة الذاتية التلقائية:** فحص هيكلية قاعدة البيانات تلقائياً بطلب `PRAGMA table_info` وإجراء التعديلات (ALTER TABLE) لضمان اتساق قواعد بيانات D1 دون الحاجة لصلاحيات كتابة مباشرة من الـ API الخارجي.
- **التشفير الآمن المدمج:** تشفير كلمات المرور باستخدام Web Crypto API (خوارزمية PBKDF2) لضمان الأمان السيبراني المتوافق مع Edge Runtime في Cloudflare.
- **تأمين مسارات الإدارة:** التحقق من هيدر `X-Admin-Token` في مسارات خادم الإدارة لمنع الوصول غير المصرح به.
- **تنظيف الحزم البرمجية:** الحفاظ على نظافة كود TypeScript عن طريق إزالة تكرار الواجهات (Interfaces) مثل `Sector` واستيرادها من مصادرها الأصلية.
